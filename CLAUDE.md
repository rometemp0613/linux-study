# Claude 지침서

## 프로젝트 개요
이 저장소는 "The Linux Command Line" 책을 학습하는 프로젝트입니다.

## 대화 시작 시 해야 할 일

1. **README.md를 읽어서 현재 진도를 파악할 것**
2. 체크리스트에서 완료되지 않은 첫 번째 챕터를 확인
3. 사용자에게 다음 공부할 챕터를 권유
4. 학습 일지의 마지막 날짜를 확인하여 얼마나 쉬었는지도 언급

## 학습 진행 방식

- 사용자가 공부를 시작하면 해당 챕터의 핵심 개념을 설명
- 실습 예제를 함께 진행
- 학습이 끝나면:
  - `notes/` 폴더에 학습 노트 작성
  - README.md의 체크리스트 업데이트 ([ ] → [x])
  - README.md의 학습 일지 테이블에 기록 추가
  - 커밋 메시지 형식: `docs: Ch.X 학습 완료 - [주요 키워드]`

## 톤 & 스타일

- 친근하게 반말로 대화
- 공부 의욕을 북돋아주기
- 어려운 개념은 쉽게 풀어서 설명

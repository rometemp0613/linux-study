# 복습 중인 용어/개념

## Ch.1~8 복습 시험 (2026-02-12) — 틀린 것 & 몰랐던 것

- **`-`로 시작하는 파일 삭제법**: `rm -- -file.txt` 또는 `rm ./-file.txt`. 따옴표로는 안 됨 — 따옴표는 셸 확장을 막는 거고, `-` 옵션 해석은 명령어 자체의 문제
- **`$()` vs `$(())`**: `$()` = 명령어 치환(명령어 실행 → 출력 결과로 대체). `$(())` = 산술 확장(수학 계산 → 결과로 대체, 정수만)
- **`rm -rf` 위험한 이유**: `-r` 재귀 삭제 + `-f` 확인 없이 강제 삭제 + 리눅스에는 휴지통이 없어서 복구 불가
- **`2>&1` 순서 문제**: `> file 2>&1` = 둘 다 file / `2>&1 > file` = stderr는 화면, stdout만 file. 핵심: 실행 시점의 stdout 위치를 따라감
- **`$((15 % 4))` = 3**: 산술 확장은 정수 연산만. `%`는 나머지 연산 (3.75가 아님!)
- **bash 풀네임**: Bourne-Again Shell. 원조 셸: sh (Bourne Shell)
- **less 조작**: `G` = 맨 끝, `g` = 맨 처음, `/단어` = 검색, `n` = 다음 결과, `q` = 나가기
- **`cp -r` 없이 디렉토리 복사**: 에러 발생. `-r`(recursive)이 있어야 디렉토리 전체 복사 가능
- **하드 링크 vs 심볼릭 링크 3가지 차이**: 원본 삭제 시(하드=유지, 심볼릭=깨짐) / 디렉토리(하드=불가, 심볼릭=가능) / 다른 디스크(하드=불가, 심볼릭=가능)

# 완벽히 암기한 용어/개념

## Ch.1~8 복습 (2026-02-13 승급)

- **cd가 내장(builtin)인 이유**: 외부 프로그램이면 자식 프로세스에서 디렉토리 변경 후 사라짐. 현재 셸 자체의 디렉토리를 바꿔야 하므로 내장 필수
- **큰따옴표 안에서 되는 확장**: `$변수`, `$(명령)`, `$((식))` — $ 계열만. 안 되는 것: `*`, `~`, `{a,b}`
- **`ls -l` 맨 앞 글자**: `d` = 디렉토리, `-` = 일반 파일, `l` = 심볼릭 링크
- **프롬프트 `$` vs `#`**: `$` = 일반 사용자, `#` = root (슈퍼유저)

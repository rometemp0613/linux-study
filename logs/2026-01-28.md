### 새로 배운 명령어
| 명령어 | 설명 | 예시 |
|--------|------|------|
| `date` | 현재 날짜/시간 표시 | `date` |
| `cal` | 달력 표시 | `cal` |
| `df` | 디스크 사용량 | `df -h` |
| `free` | 메모리 사용량 (Linux) | `free` |
| `exit` | 터미널 종료 | `exit` 또는 `Ctrl+D` |

### 새로 배운 단축키
| 단축키 | 기능 |
|--------|------|
| `Ctrl+C` | 실행 중인 명령 중단 |
| `Ctrl+D` | 터미널 종료 |
| `Ctrl+L` | 화면 클리어 |
| `↑` / `↓` | 명령어 히스토리 탐색 |

### 메모 / 느낀 점
- Shell = 키보드 명령어를 OS에 전달하는 프로그램
- bash = Bourne-Again Shell
- macOS 터미널에서도 셸 명령어는 Ctrl 사용 (Cmd 아님!)
- 터미널 앱 조작(복사/붙여넣기)만 Cmd 사용

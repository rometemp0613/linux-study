# Chapter 1: What Is the Shell?

## 핵심 개념

### Shell이란?
- **셸(Shell)**: 키보드 명령어를 받아서 운영체제에 전달하는 프로그램
- 대부분의 Linux 배포판은 GNU 프로젝트의 **bash**를 기본 셸로 사용
- **bash** = **B**ourne-**A**gain **SH**ell (Steve Bourne이 만든 원조 Unix 셸 `sh`의 개선판)

### Terminal Emulator (터미널 에뮬레이터)
- GUI 환경에서 셸과 상호작용하기 위한 프로그램
- KDE: `konsole`
- GNOME: `gnome-terminal` (보통 메뉴에서 "Terminal")

---

## Shell Prompt (셸 프롬프트)

```
[me@linuxbox ~]$
```

구성 요소:
- `me` : 사용자 이름
- `linuxbox` : 컴퓨터(호스트) 이름
- `~` : 현재 작업 디렉토리 (홈 디렉토리)
- `$` : 일반 사용자 표시 (`#`이면 root/superuser)

---

## 기본 명령어

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `date` | 현재 날짜와 시간 표시 | `Fri Feb 2 15:09:41 EST 2018` |
| `cal` | 현재 달 달력 표시 | 달력 출력 |
| `df` | 디스크 여유 공간 표시 | 파일시스템별 사용량 |
| `free` | 메모리 사용량 표시 | 총/사용/여유 메모리 |
| `exit` | 터미널 세션 종료 | 또는 `Ctrl+D` |

---

## 유용한 기능

### Command History (명령어 히스토리)
- **위쪽 화살표(↑)**: 이전 명령어 불러오기
- **아래쪽 화살표(↓)**: 다음 명령어로 이동
- 대부분의 Linux는 최근 1000개 명령어를 기억

### Cursor Movement (커서 이동)
- **좌/우 화살표**: 명령줄에서 커서 이동
- 명령어 편집이 쉬워짐

### Copy & Paste (복사/붙여넣기)
- **복사**: 마우스 왼쪽 버튼으로 드래그 (자동 복사됨)
- **붙여넣기**: 마우스 가운데 버튼 클릭
- 주의: `Ctrl+C`, `Ctrl+V`는 터미널에서 다른 의미로 사용됨!

---

## Virtual Console (가상 콘솔)
- GUI 뒤에서 실행되는 터미널 세션들
- `Ctrl+Alt+F1` ~ `Ctrl+Alt+F6`: 가상 콘솔 접근
- `Alt+F7`: 다시 그래픽 데스크탑으로 복귀
